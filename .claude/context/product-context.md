# Product Context

## Target Users
- Home cooks who want quick, structured recipes from YouTube
- Viewers who prefer step-by-step recipe cards vs. long videos

## Core Value
- Turn YouTube cooking videos into clean, followable recipes
- Provide translation and curation for faster cooking decisions

## Primary Use Cases
- Search or browse recipe categories
- Watch video while following a recipe card
- Save or cache recipes for quick access

## MVP Constraints
- iOS 17+ only
- English output by default
- YouTube compliance (embedded playback)
