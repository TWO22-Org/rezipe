# Project Overview

## Summary
ChefStream (Rezipe) is a recipe-first YouTube experience for iOS that extracts structured recipes from cooking videos.

## Key Features
- Curated discovery + search for recipe videos
- Embedded YouTube playback (IFrame-compliant)
- Recipe extraction and translation into a structured card
- Cached recipes for quick repeat access

## Integrations
- YouTube Data API v3 + IFrame Player API
- Supabase (Postgres, Edge Functions, Auth)
- LLM service for extraction + translation

## Current State
- MVP development phase
- Documentation and agent workflows defined
